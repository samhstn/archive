# Auth Riot Todo app

### What

A todo application with login and sweet todo management

### Why

To practice riot, postgres and hapi technologies

### How

To get started run:

```
git clone https://github.com/shouston3/auth_riot_todo.git && cd auth_riot_todo
```

Install the dependencies:
```
npm install
```

Start the server:

```
npm start
```
